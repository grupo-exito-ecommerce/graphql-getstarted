# GetStarted API

Para importar en su componente:

```js
manifest.json

"dependencies": {
  "exito.graphql-getstarted": "1.x"
}

```

## Uso

Se deben de proveer los siguientes datos.

```js
user(name: String, pwd: String)
```

Donde se retornaran los mismos datos

```js
name: String;
pwd: String;
```
